# musterisegmentasyonu
Bu proje müşteri segmentasyonu için K-Means, Hiyerarşik ve Affinity Propagation algoritmalarını kullanır.

## Kullanılan Teknolojiler
- Python
- Pandas, Numpy, Seaborn, Matplotlib, Sklearn

## Çalıştırma
1. Python 3.10+ kurulumu
2. Gereksinimler: pip install -r requirements.txt
3. Çalıştır: python main.py

## Sonuçlar
- 5 kümeye ayrılmış müşteri dağılımları
- Silhouette ve Davies-Bouldin skorları
